# Zerofour
## Install
### NodeJS npm
Install Nodejs and npm for your system
## Install the proyect  
```
- npm install -g @angular/cli   If you have any error try to run it with root permision
- git clone https://github.com/dlowTux/ZeroFour
- cd ZeroFour
- npm i
```
## Run
```
- ng serve --open
```
